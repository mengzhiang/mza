<static> i18n.add(langObj)
功能:
添加语言包对象 
Jet().$package(function(J){
	// 添加一个名字为"zh-cn"中文语言包
	J.i18n.add({
		"zh-cn":{
			// 数字
			number:"ddd,ddd.dd",
			// 货币
			currency:"￥ddd,ddd.dd元",
			// 日期
			date: "yyyy-MM-dd",
			// 长日期
			longDate:"yyyy年MM月dd日",
			// 时间
			time: "hh小时mm分ss秒",
			// 文本
			text: {
				helloWorld:"你好,世界!",
				test: "测试",
				onlyDefaultHas: "只有默认语言才有的...",
				about: "Jet 框架, 开发者: 于涛!"
			}	
		}
	});
};
参数:
{Object} langObj
要添加的语言包对象
